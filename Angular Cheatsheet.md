# Angular Cheatsheet
- Check installation: `ng --version`
- Create new app: `ng new <NAME>` (No routing, CSS as stylesheet)
- Open app in browser: `ng serve -o`
- Create component: `ng generate component <PATH/TO/COMPONENT>`
